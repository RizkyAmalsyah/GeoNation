## Cara Menggunakan

Untuk menjalankan proyek ini di lingkungan lokal, Anda dapat mengikuti langkah-langkah berikut:

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/RizkyAmalsyah/GeoNation.git
   cd GeoNation
   composer install
   cp .env.example .env
   php artisan migrate
   php artisan serve
